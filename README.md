# 🎮 Software Development 02 Project

## 📌 Project Overview
This project is a **Esports Tournament Management System** developed using **PHP & MySQL**.  
It allows management of:
- Players 👤
- Teams 🏆
- Matches ⚔️
- Tournaments 🗓️
- Results 📊

---

## 🚀 Features
- Add & manage players
- Create & manage teams
- Organize tournaments
- Match scheduling system
- Store and view results

---

## 🛠️ Technologies Used
- **PHP**
- **MySQL**
- **CSS**
- **HTML**
